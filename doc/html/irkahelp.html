No examples

Description

This function executes the Iterative Rational Krylov Algorithm (irka) as proposed by Gugergin and Beattie in [1].

The irka iteration is conducted to search for an optimal set of shifts in Krylov subspace-based model reduction. If irka converges, then the reduced model is known to be a local optimum with respect to the H2 norm of the error.
